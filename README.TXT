# Java Gym Registration System

This is a simple desktop application built in Java that lets you manage gym memberships. It uses object-oriented programming concepts like inheritance and includes a graphical user interface (GUI) built with Java Swing.

## Features

- GUI-based interface for easy use
- Add and manage Regular and Premium members
- Store and view member details
- Saves data to a text file
- Organized code using OOP principles

## Tools & Technologies

- Java
- Java Swing for GUI
- File handling with .txt files
